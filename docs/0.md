# 前言

佛家与道家的典籍卷帙浩繁，文辞讲究，义理深奥。然而对于初学者而言，其语言往往过于艰涩，不易入门。更遗憾的是，像《华严经》《瑜伽师地论》等许多重要经典，至今仍缺乏完整的现代汉语译本。笔者在学习过程中发现，市面上或网络上能获取的资料，大多仅收录原文及少量注疏，这无疑为初学者深入理解带来了不小的障碍。

经典本身是最珍贵的学习资源，若因文字隔阂而却步，实为可惜。随着信息时代的推进，人工智能技术迅速发展，笔者作为一名计算机工程师，遂萌生借助AI之力尝试对这些典籍进行翻译与整理的念头。于是决定将翻译内容陆续整理成书或博客，既是对个人学习的记录，也希望能为后来的学习者略尽绵薄，让更多人能够更便捷地亲近经典、深入经藏。

因为时间精力有限，笔者仅用工程化手段做完了全部的翻译和章节整理，并未校对，内容和翻译皆可能有误，后续可能继续更新校对的版本，读者敬请注意。典籍翻译本应该是一件极为严肃和认真的事情，但笔者此次只是将现有古文以便利方式转译为白话文方便阅读使用，让普通人不至于被文字章句所卡住，并不是真正意义上的翻译，读者应更加注重原文而非译文，经文原本是经无数高僧大德字字揣摩而来，质量远非现代人工或AI译本可比。为了便利阅读和传播，笔者会将所有内容开源和免费，放在gitee和github网站上作为托管。
